# Asset Factory (P-MKT-04)

Cel: **koszt produkcji / kanał → 0** przy kolejnych surfaces.

## Rytm

1 dzień zdjęciowy / 2 tygodnie (lub reuse mastera jeśli świeży &lt;30d).

## Output obowiązkowy (`MKT/YYYY-WW/`)

| Plik | Format | Użycie |
|------|--------|--------|
| `master_reel_9x16.mp4` | 9:16 | Ads A + organic FB/IG |
| `feed_1x1.mp4` | 1:1 | Feed ads/organic |
| `carousel_1.png` … `_3.png` | square | Ads B |
| `testimonial.jpg` | photo | Ads C |
| `tt_hook_15s.mp4` | 9:16 ≤15s | TikTok |
| `NOTES.md` | text | hipoteza, UTM, CTA |

## Definition of Done WW

- [ ] Wszystkie pliki powyżej  
- [ ] Prawdziwy bus (nie stock) w masterze  
- [ ] NOTES z `utm_campaign` + listą kanałów  
- [ ] Tego samego dnia: Commander HITL **lub** Ads upload **lub** oba  
- [ ] TikTok hook zaplanowany (patrz TIKTOK-ORGANIC)

## Odrzut

Asset bez CTA/UTM w NOTES · AI-only bus bez real photo · osobne kreacje per kanał bez mastera.
