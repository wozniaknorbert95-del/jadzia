# D0.15 ADR — Hub IA navigation (5 primary)

**Status:** Accepted  
**Date:** 2026-07-18  
**Related:** D0.6 phone hub; UX-BRIEF IA max 5

## Decision

Primary destinations (desktop ≡ mobile):

1. Start (Home)  
2. Marketing  
3. Analityka  
4. Agenci  
5. Ustawienia  

**Audyt** = secondary — reachable from Ustawienia (link) and/or Więcej sheet; never a 6th peer primary tab.

## Consequences

- Bottom nav 5th slot = Ustawienia (not Więcej-as-only-settings-hide).  
- Więcej sheet retained only if needed for Audyt overflow; preferred: Audyt entry on Settings.  
- `aria-current="page"` on active primary nav control.
