# D0.5 ADR — COI Commander Hosting

**Status:** Accepted  
**Date:** 2026-07-09

## Context

COI Commander needs authenticated control plane for marketing, analytics, agents. Options: greenfield SPA, embed in VCMS, extend agent-os-ui.

## Decision

**Greenfield SPA** in `jadzia-core/commander-ui/`, served by jadzia-core FastAPI at `/commander/`.

## Rationale

- JWT proxy already on jadzia-core :8000
- COI ≠ engineering HITL (agent-os-ui)
- VCMS = governance only (deep link)
- Single deploy unit with spine APIs

## Consequences

- CORS already allows localhost:5173 for dev
- Prod: reverse proxy `/commander` → jadzia static
- ADR reversible if VCMS absorbs UI later
