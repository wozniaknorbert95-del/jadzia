# D0.10 — Audit Log Full Spec (N2)

## Storage

- SQLite `commander_audit_log`, INSERT-only
- No UPDATE/DELETE on application path

## Retention

- 24 months (GDPR-defined)
- Export before purge

## Reader roles

- Dowódca, Delegat: read
- Viewer: no access

## Tamper-evidence

Hash-chain: `row_hash = SHA256(prev_hash + canonical_json)`

## Required fields

`id, ts, actor_id, actor_role, action, source, target_type, target_id, before_json, after_json, reason, risk_tier, session_id, prev_hash, row_hash`
