# D0.16 — Session chrome (JWT / TG code)

**Status:** Spec  
**Date:** 2026-07-18

## States

| State | UI |
|-------|-----|
| logged-out | Auth panel expanded: TG hint + JWT fallback + Zapisz |
| valid | Auth panel **collapsed**: status chip „Zalogowano” + button „Sesja” to expand; no primary token field in first viewport |
| 401 | Expand auth; toast PL: „Sesja wygasła — Telegram /commander lub wklej token” |
| recover | TG `?code=` exchange OR paste JWT; strip URL params |

## Rules

- Never show filled JWT as primary chrome when session valid.  
- Logout clears token and expands panel.  
- Legacy `?jwt=` still works; prefer `?code=`.
