# D0.11 ‚Äî HITL‚ÜíHOTL Graduation Thresholds (N3)

Per action-type (`fb_post_approve`, `lead_score`, ...):

```
IF approved_without_edit >= N (default 20)
   AND override_rate < X% (default 5%) over rolling 30d
   AND confidence_avg >= threshold
THEN graduate to HOTL (auto with notify)
```

Revert on override spike. UI badge: HITL / HOTL / auto.

## API

`POST /api/v1/commander/feedback` captures rejection/correction and increments counters.
