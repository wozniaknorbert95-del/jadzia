# D0.13 — Multi-Role Authz (N7)

## JWT claims

`role`: `dowodca` | `delegat` | `viewer`  
`scopes[]`: explicit scope list (optional override)

## Server-side scopes

| Role | Scopes |
|------|--------|
| Dowódca | `*` |
| Delegat | `marketing:approve`, `marketing:publish`, `queue:act`, `leads:act` |
| Viewer | `*:read` |

Backend returns **403** on scope violation. UI hide is not security.

## Block escalate-through-UI

Delegat cannot assign Dowódca role.
